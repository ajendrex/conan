API Reference Guide for Conan's Python Bindings

undirected_graph class

undirected_graph

  • Constructors
    • undirected_graph ()
      • Effects
      Construct an empty undirected graph.

    • undirected_graph (num_vertices)
      • num_vertices: Number of vertices (int)
      Effects
      Construct an undirected graph with num_vertices vertices and zero edges.

    • undirected_graph (adj_matrix)
      • adj_matrix: Adjacency matrix (list)
      Effects
      Construct an undirected graph from the adjacency matrix adj_matrix.

  • Manipulation
    • vertices ()
      • Effects
      Returns the list of vertices.

    • edges ()
      • Effects
      Returns the list of edges.

    • add_vertex (name = '')
      • name: Vertex name (string)
      Effects
      Add a new vertex to the graph and returns the vertex object.

    • add_edge (v1, v2)
      • v1: Source vertex (vertex)
      • v2: Target vertex (vertex)
      Effects
      Add a new edge to the graph, linking v1 and v2 and returns the edge object.

    • remove_vertex (v)
      • v: Vertex to be removed (vertex)
      Effects
      Remove vertex v from graph and all its incident edges.

    • remove_edge (e)
      • v: Vertex to be removed (vertex)
      Effects
      Remove edge e from graph.

    • remove_edge (v1, v2)
      • v1: Source vertex (vertex)
      • v2: Target vertex (vertex)
      Effects
      Remove edge between v1 and v2 from graph.

    • get_vertex (v_id)
      • v_id: Vertex id (int)
      Effects
      Returns the vertex object whose id is v_id.

    • get_edge (v1, v2)
      • v1: Source vertex (vertex)
      • v2: Target vertex (vertex)
      Effects
      Returns the edge object that links v1 and v2. If that edge does not exists, it raises a RuntimeError exception.

    • get_adj_matrix ()
      • Effects
      Returns the graph's adjacency matrix as a list of lists.

  • Properties
    • num_vertices ()
      • Effects
      Returns the number of vertices of the graph.

    • num_edges ()
      • Effects
      Returns the number of edges of the graph.

    • degree_distribution ()
      • Effects
      Returns a degree_distribution object that you can use to retrieve properties of the graph's degree distribution.

    • shortest_paths ()
      • Effects
      Returns a shortest_path object that you can use to retrieve graph's and vertex's average shortest path, vertex's eccentricity, and graph's radius, diameter and center.

    • betweenness_centrality ()
      • Effects
      Returns a betweenness_centrality object that you can use to retrieve graph's, vertex's and edge's betweenness centrality measures.

    • connectance ()
      • Effects
      Returns the connectance of the graph.

    • distance (source_vertex, target_vertex)
      • source_vertex: Source vertex (vertex)
      • target_vertex: Target vertex (vertex)
      Effects
      Returns the distance between source_vertex and target_vertex.

    • asp ()
      • Effects
      Returns the graph's average shortest path.

    • asp_normalized ()
      • Effects
      Returns the graph's average shortest path normalized by the average shortest path of a random network with same number of vertices and number of edges.

    • avg_clustering ()
      • Effects
      Returns the graph's average clustering coefficient.

    • avg_clustering_normalized ()
      • Effects
      Returns the graph's average clustering coefficient normalized by the average clustering coefficient of a random network with same number of vertices and number of edges.

    • entropy ()
      • Effects
      Returns the topological entropy of the graph.

    • fractal_dimension ()
      • Effects
      Returns the fractal dimension of the graph. This is calculated using a box-counting-like method.

  • Input/Output
    • read_dot (input_filename)
      • input_filename: The name of a DOT file to read from (string)
      Effects
      The instance contains the graph stored in input_filename.

    • read_csv (input_filename, fields_order = 'stw')
      • input_filename: The name of a CSV file to read from (string)
      • fields_order (string): This string defines the order in which the different fields (columns) are ordered into the file. The fields are:
        • s: source vertex (required)
        • t: target vertex (required)
        • w: edge weight (optional)
      Effects
      The instance contains the graph stored in input_filename.

    • read_adj_matrix_file (input_filename, separator_char = ' ')
      • input_filename: The filename to read from (string)
      • separator_char: The character that separates columns. It must be only one character (string)
      Effects
      The instance contains the graph stored in input_filename.

    • read_graphml (input_filename)
      • input_filename: The name of a GraphML file to read from (string)
      Effects
      The instance contains the graph stored in input_filename.

    • read_gml (input_filename)
      • input_filename: The name of a file in GML format to read from (string)
      Effects
      The instance contains the graph stored in input_filename.

    • read_pajek (input_filename)
      • input_filename: The filename to read from (string)
      Effects
      The instance contains the graph stored in input_filename.

    • write_graphml (output_filename)
      • output_filename: The name of the file to write to (string)
      Effects
      Store the network in output_filename in GraphML format.

    • write_dot (output_filename)
      • output_filename: The name of the file to write to (string)
      Effects
      Store the network in output_filename in DOT format.

    • write_gml (output_filename)
      • output_filename: The name of the file to write to (string)
      Effects
      Store the network in output_filename in GML format.

    • write_csv (output_filename, fields_order = 'stw')
      • output_filename: The name of the file to write to (string)
      • fields_order (string): This string defines the order in which the different fields (columns) are ordered into the file. The fields are:
        • s: source vertex (required)
        • t: target vertex (required)
        • w: edge weight (optional)
      Effects
      Store the network in output_filename in CSV format.

    • write_pajek (output_filename)
      • output_filename: The name of the file to write to (string)
      Effects
      Store the network in output_filename in the file format used by the program Pajek.

    • write_adj_matrix_file (output_filename)
      • output_filename: The name of the file to write to (string)
      Effects
      Store the network in the plain text adjacency matrix file output_filename.

    • save_as (image_filename, program = 'neato')
      • image_filename: The name of the file to write to (string)
      • program: The name of GraphViz program to draw the network (string)
      Effects
      Save a drawing of the network in image_filename.

    • view (program = 'neato')
      • program: The name of GraphViz program to draw the network (string)
      Effects
      Draw and show the network in a GTK+/Qt window.

  • Subgraphs, modules and communities
    • subgraph (l)
      • l: A list of vertices (list)
      Effects
      Returns a subgraph from the given list of vertices. The subgraph is itself a graph object.

    • components ()
      • Effects
      Returns a list of components. A component is a list of vertices.

    • newman_communities (check_quality_fcn)
      • Effects
      Returns a list of modules. A module is a list of vertices. To compute modules, the function maximizes the modularity of the network. For further reference see Newman (2006).

    • newman_two_communities ()
      • Effects
      Returns a list of only two modules. A module is a list of vertices. To compute modules, the function maximizes the modularity of the network. The two modules are the first division the algorithm founds. For further reference see Newman (2006).

    • minimum_spanning_tree ()
      • Effects
      Returns the minimum spanning tree as a graph object.

    • nearest_neighbour_ring (start_vertex_id = 0)
      • start_vertex: The vertex from which the construction of the nearest neighbour will begin (vertex)
      Effects
      Returns the nearest neighbour ring as a graph object. Rings are a kind of graphs in which every vertex is connected to only two other vertices.

  • Transformations
    • complement ()
      • Effects
      All the pair of vertices that were not edges in g are edges in the returned graph and viceversa.
      Complexity
      O(N²)

    • booleanize ()
      • Effects
      All edges have a weight of 1.

    • randomize_edges (p, maintain_connected = False)
      • p: Probability of changing a edge by a random edge (decimal)
      • maintain_connected: Whether to force the algorithm to maintain the network connected (bool)
      Effects
      Change each edge by a random each with a probability p.

    • boxcounting (boxsize)
      • boxsize: The size of boxes used for grouping vertices together (decimal)
      Effects
      Returns the graph object resulting from a box-counting renormalization process for a certain box size.
