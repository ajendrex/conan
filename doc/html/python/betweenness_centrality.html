API Reference Guide for Conan's Python Bindings

undirected_betweenness_centrality and directed_betweenness_centrality classes

betweenness_centrality

  • Constructors
    • betweenness_centrality (g)
  • vertex_centrality (v)
    • v: A vertex for which to compute the absolute vertex centrality (vertex)
    Effects
    Compute the absolute vertex centrality of vertex v.

  • vertex_relative_centrality (v)
    • v: A vertex for which to compute the relative vertex centrality (vertex)
    Effects
    Compute the relative vertex centrality of vertex v.

  • edge_centrality (e)
    • e: An edge for which to compute the absolute edge centrality (vertex)
    Effects
    Compute the absolute edge centrality of edge e.

  • central_point_dominance ()
    • Effects
    Compute the central point dominance of the graph. This is a measure of how centralised/decentrilised the network is.
