API Reference Guide for Conan's Python Bindings

undirected_degree_distribution and directed_degree_distribution classes

degree_distribution

  • Constructors
    • degree_distribution (g)
      • Effects
      Construct a degree distribution object for a network g.

  • P (k)
    • Effects
    Returns the probability for a certain degree k.

  • vertices (k)
    • Effects
    Returns a list of vertices with a degree k.

  • non_zero_degrees ()
    • Effects
    Returns the list of degrees that have a probability different than zero.

  • gamma ()
    • Effects
    Returns the value of the power-law decay exponent.

  • exp_decay ()
    • Effects
    Returns λ for the exponential fit A * e (λ * k) + C.

  • slope ()
    • Effects
    Returns the slope for the linear fit.

  • r (fit_type)
    • fit_type (string): Possible values are: 'linear', 'exponential', and 'power'.

    Effects
    Returns the Pearson's correlation coefficient for a certain fit.

  • rr ()
    • fit_type (string): Possible values are: 'linear', 'exponential', and 'power'.

    Effects
    Returns the squared Pearson's correlation coefficient for a certain fit.

  • entropy ()
    • Effects
    Returns the statistical entropy of the distribution.

  • best_fit ()
    • Effects
    Returns a string that allows to identify which was the best fit, based on the values of the squared Pearson's correlation coefficients. Possible return values are: 'linear', 'exponential', and 'power'.
